JavaScript Array Concat & Join Tutorial

Concat Array

The concat() method is used to merge two or more arrays together. This method does not alter the original arrays passed as arguments.

eg: var x = ["Ram","Arman","Raghav","Manav"]
var y = ["Riya","Diya","Amit"]
x.concat(y);


Join Array

The join() method is used to join the elements of an array into a string. The elements of the string will be separated by a spcified separator and its default calue is a comma(,).

eg: var x = ["Ram","Arman","Raghav","Manav"]
x.join("-");

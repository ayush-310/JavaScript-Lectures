JavaScript IndexOf & LastIndexOf Tutorial

IndexOfArray

The indexOf() method is used to find the index of the first occurrence of the search element provided as the argument to the function.

eg: var x = ["Ram","Arman","Raghav","Manav"]
indexOf("Search item" , start);


LastIndexOfArray

The lastIndexOf() method is used to find the index of the last occurrence of the search element provided as the argument to the function.

eg: var x = ["Ram","Arman","Raghav","Manav"]
indexOf("Search item" , start);

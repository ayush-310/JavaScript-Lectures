JavaScript insertAdjacentElement & insertAdjacentHTML Methods

insertAdjacentElement()

insertAdjacentElement() method inserts the specified element at the specified position. The legal values for this position are.

Syntax

node.insertAdjacentElement(position,element)

insertAdjacentHTML()

The DOM insertAdjacentHTML() method is used to insert a text as HTML file to a specified position. This method is used to change or add text as HTML. Return Value : This will return the page with the specified change. There are four legal position values that can be used.

Syntax

node.insertAdjacentHTML(specify-position,text-to-enter);

insertAdjacentText()

It provide text at one of the following

  • afterbegin
  • afterend
  • beforebegin
  • beforeend
Syntax

node.insertAdjacentText(position,text);

JavaScript addEventListener Method Tutorial

DOM Normal Function

Document.getElementByid(ID).onclick = functionName;

DOM addEventListener Methods:

Document.getElementByid(ID).addEventListener("click",functionName);

Document.getElementByid(ID).addEventListener("click",function(){
statement;
});

DOM removeEventListener Methods:

Document.getElementByid(ID).removeEventListener("click",functionName);

Document.getElementByid(ID).removeEventListener("click",function(){
statement;
});

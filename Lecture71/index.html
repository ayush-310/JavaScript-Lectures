JavaScript replaceChild & removeChild Methods

Replace Child

The replaceChild() method in HTML DOM is USE to replace a child node with a new node within the given parent node.

parentNode.replaceChild(newChild, oldChild);

Remove Child

The reChild() method in HTML DOM is USE to remove a sprcified child node of the given element. It returns the remove node as a node object or null if the node doesn't exist.

node.removeChild(child);

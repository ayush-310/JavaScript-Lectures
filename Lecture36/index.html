JavaScript Array Shift & Unshift Tutorial

Shift Array

The shift() method removes the first element of the array tus reducing the size of the original array by 1.

eg: var x = ["Ram","Arman","Raghav","Manav"]
x.shift();


Unshift Array

The unshift() method is used to add one or more elements to the beginning of the given array. This function increases the length of the existing array by the number of elements added o the array.

eg: var x = ["Ram","Arman","Raghav","Manav"]
x.unshift();
